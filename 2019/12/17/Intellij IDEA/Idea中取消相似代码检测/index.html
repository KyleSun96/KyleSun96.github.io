IDEA中取消相似代码检测

Author: Kyle Sun Date: 十二月 17, 2019  15:22:12 Category: Intellij IDEA

问题:
在IDEA中根据设置的不同,有些代码页,当代码重复比较多时,会出现灰色或黄色的波浪线

解决方法:
如果不想让它出现波浪线的话，可以在File–>Settings–>Editor–>
–>Inspections–>General–>Duplicated Code 把方框里的勾取消即可
