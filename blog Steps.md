==Blog Steps==

***
1. MyBlog目录 git bash here
2. hexo new "XXXX"
3. Typora编辑
4. hexo clean
5. hexo g		 (生成)
6. hexo s		 (start/sever 开启,可以本地预览)
7. http://localhost:4000/  本地查看
8. Ctrl+C    ==(断开)==
9. hexo d   ==(部署到远端)== 

***
PS:
hexo clean
hexo generate   (若要本地预览就先执行 hexo server)
hexo deploy

快捷命令：
hexo g == hexo generate
hexo d == hexo deploy
hexo s == hexo server
hexo n == hexo new
